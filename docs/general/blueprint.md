# **App Name**: MyBasket Lite

## Core Features:

- Product Listing: Display a list of grocery products with images, names, and prices.
- Shopping Cart: Allow users to add products to the cart and view the cart total.
- Order Review: Provide a checkout process for users to review their cart, and view their completed orders
- Personalized Recommendations: AI powered suggestions based on shopping cart
- AI powered grocery suggestions: LLM analyzes current cart and acts as a "tool" suggesting groceries to the user they might need or enjoy based on existing cart content.
- Sample Data: Include sample grocery products and images

## Style Guidelines:

- Primary color: Soft, desaturated green (#90EE90) to evoke a sense of freshness and natural products.
- Background color: Very light, almost white, desaturated green (#F0FFF0) to provide a clean and airy feel.
- Accent color: A slightly more saturated yellow-green (#ADFF2F) to highlight important elements and CTAs.
- Clean and readable sans-serif font for product names, descriptions, and UI elements.
- Simple, modern icons for categories, cart, and user actions.
- Grid-based layout for product listings, ensuring a neat and organized display.
